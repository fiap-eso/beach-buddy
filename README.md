## Tutorial para rodar o projeto

- Instalar os pacotes do npm
```bash
npm install
```

- Iniciar servidor local
```bash
npm start
```

- Acessar via browser na url: http://localhost:5173/beach-buddy

## Versoes utilizadas

- Node: v20.11.0
- NPM: 10.2.4
